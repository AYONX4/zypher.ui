# Getting Started with ZypherUI

Follow these simple steps to start using ZypherUI components in your project.

## Prerequisites

ZypherUI is built on [Tailwind CSS v4](https://tailwindcss.com/docs/installation/framework-guides/nextjs), which must be installed in your project before continuing.

## 1. Install Utilities

Many components use the `cn` utility function for conditional class names. Install it with:

<Tabs items={["bun", "npx", "pnpm"]}>
 
```bash tab="bun"
bunx shadcn@latest add https://ui.codesnippet.dev/r/utils.json 
```
 
```bash tab="npx"
npx shadcn@latest add https://ui.codesnippet.dev/r/utils.json
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://ui.codesnippet.dev/r/utils.json
```

</Tabs>

## 2. Add Components

Once utilities are installed, you can add individual components using the shadcn CLI:

<Tabs items={["bun", "npx", "pnpm"]}>
 
```bash tab="bun"
bunx shadcn@latest add https://ui.codesnippet.dev/r/card-05.json
```
 
```bash tab="npx"
npx shadcn@latest add https://ui.codesnippet.dev/r/card-05.json
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://ui.codesnippet.dev/r/card-05.json
```

</Tabs>

> **Note:** While code snippets can be copied manually, we strongly recommend using the CLI for installing components to ensure all dependencies are properly included.

### Using the Component

After installation, import and use the component in your page:

```jsx
import Card05 from "@/components/codesnippetui/card-05";

export default function Page() {
    return <Card05 />;
}
```

## 3. Working with Blocks

Blocks are comprehensive components that include multiple files. They must be installed using the CLI:

<Tabs items={["bun", "npx", "pnpm"]}>
 
```bash tab="bun"
bunx shadcn@latest add https://ui.codesnippet.dev/r/ai-card-generation.json
```
 
```bash tab="npx"
npx shadcn@latest add https://ui.codesnippet.dev/r/ai-card-generation.json
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://ui.codesnippet.dev/r/ai-card-generation.json
```

</Tabs>

### Using Blocks

Once installed, blocks can be used like regular components:

```jsx
/**
 * AICardGeneration includes 5 additional components
 * that are automatically installed with the CLI.
 */
import AICardGeneration from "@/components/codesnippetui/AICardGeneration";

export default function Page() {
    return <AICardGeneration />;
}
```

## 4. Dependencies

### Core Dependencies

CodeSnippetUI uses:

- [Lucide Icons](https://lucide.dev/guide/installation) - For component icons
- [shadcn/ui](https://ui.shadcn.com/) - For base components

These are automatically installed when using the CLI.

### Component-Specific Dependencies

Some components require additional libraries, which are listed on each component's documentation page. Be sure to install these dependencies to ensure proper functionality.

## 5. Monorepo Setup

When working in a monorepo, use the `-c` flag to specify your workspace path:

<Tabs items={["bun", "npx", "pnpm"]}>
 
```bash tab="bun"
bunx shadcn@latest add https://ui.codesnippet.dev/r/card-05.json -c ./apps/www
```
 
```bash tab="npx"
npx shadcn@latest add https://ui.codesnippet.dev/r/card-05.json -c ./apps/www
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://ui.codesnippet.dev/r/card-05.json -c ./apps/www
```
